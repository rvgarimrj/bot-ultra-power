## Apps Pesquisados

### 1. WikiScroll - TikTok-Style Knowledge Feed (TOP PICK do dia)

- **Source:** Show HN #10 - "Scroll Wikipedia like TikTok" (quack.sdan.io), 330 points, 88 comments
- **O QUE FAZ:** Interface de swipe vertical (TikTok-style) com fatos interessantes do Wikipedia API, resumidos por AI em cards de 30 segundos
- **QUEM PRECISA:** Pessoas que querem aprender mas tem attention span curto (treinado por short-form video)
- **DOR:** Wikipedia/conteudo educacional e chato de consumir no formato tradicional
- **MVP Scope:**
  - Vertical swipe interface (TikTok-style)
  - Pull random "interesting facts" from Wikipedia API
  - AI summarizes into 30-second readable cards
  - Category filters (Science, History, Tech, etc.)
  - Share button for social
- **SCORE:** 43/50
  - Pain: 5/5 (x3 = 15) - Universal, everyone wants to learn
  - Pay: 3/5 (x3 = 9) - Freemium model possible
  - Buildable: 5/5 (x2 = 10) - Wikipedia API is free, simple UI
  - Differentiation: 4/5 (x1 = 4) - Focus on TOPICS beyond Wikipedia
  - Market: 5/5 (x1 = 5) - Billions use TikTok
- **Verdict:** BUILD
- **Concorrentes:** Original WikiTok (VSCode extension), quack.sdan.io (uses Gemini), Random Wikipedia apps
- **Market Size (TAM):** Bilhoes (edtech + social learning)

### 2. LocalPDF - Privacy-First PDF Toolkit

- **Source:** Show HN #29 - "Pdfwithlove - PDF tools that run 100% locally", 177 points, 131 comments
- **O QUE FAZ:** Merge, split, compress PDFs e convert images - ALL processing in browser (WebAssembly/pdf-lib)
- **QUEM PRECISA:** Pessoas que precisam de PDF tools mas temem upload de documentos sensiveis (bank statements, contracts, IDs)
- **DOR:** Privacy concerns reais - comentario HN: "I see a webapp and I immediately assume everything I do in this app is exfiltrated"
- **MVP Scope:**
  - Merge PDFs (most common need)
  - Split PDFs
  - Compress PDFs
  - Convert images to PDF
  - ALL processing in browser (WebAssembly/pdf-lib)
  - Clear "Never Uploaded" badge/messaging
- **SCORE:** 43/50
  - Pain: 5/5 (x3 = 15) - Universal need, privacy growing concern
  - Pay: 4/5 (x3 = 12) - Advanced features (OCR, batch processing)
  - Buildable: 4/5 (x2 = 8) - pdf-lib exists, WebAssembly
  - Differentiation: 3/5 (x1 = 3) - Crowded but privacy angle strong
  - Market: 5/5 (x1 = 5) - Everyone uses PDFs
- **Verdict:** BUILD
- **Concorrentes:** iLovePDF (cloud-based, privacy), Smallpdf (cloud, paid), Ghostscript (CLI), pdfwithlove
- **Market Size (TAM):** Bilhoes (document tools market)

### 3. IdeaVault - Simple Idea Storage & Discovery

- **Source:** Reddit r/SideProject - "Great Wall of Ideas" (greatwallofideas.xyz), trending
- **O QUE FAZ:** Submit ideias anonimamente, browse por categoria, upvote/comment, "Claim" uma ideia (mark as being built)
- **QUEM PRECISA:** Indie hackers, builders com "someone should build this" moments
- **DOR:** Ideias morrem em notes apps, nenhum lugar central para store/discover startup ideas
- **MVP Scope:**
  - Submit idea anonymously (title + description + category)
  - Browse ideas by category (SaaS, Mobile, AI, etc.)
  - Upvote/comment system
  - "Claim" an idea (mark as being built)
  - Weekly digest of top ideas
- **SCORE:** 31/50
  - Pain: 3/5 (x3 = 9) - Nice-to-have, not urgent
  - Pay: 2/5 (x3 = 6) - Hard to monetize
  - Buildable: 5/5 (x2 = 10) - Super simple CRUD app
  - Differentiation: 3/5 (x1 = 3) - Could add AI to enhance/validate
  - Market: 3/5 (x1 = 3) - Niche (indie hackers/builders)
- **Verdict:** MAYBE - Easy to build but hard to monetize
- **Concorrentes:** Great Wall of Ideas (basic), Indie Hackers Ideas Database (complex), Notes apps (private)
- **Market Size (TAM):** Centenas de milhoes (indie hacker/builder niche)
