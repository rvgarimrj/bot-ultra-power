## Apps Pesquisados

### 1. ToolMatch AI (TOP PICK do dia)

- **Source:** OECD SME Report, McKinsey State of AI, Reddit, Indie Hackers
- **O QUE FAZ:** Quiz interativo que pergunta seu problema/necessidade de negocio e recomenda ferramentas de IA especificas (nao e um directory passivo)
- **QUEM PRECISA:**
  - SMB owners, solopreneurs, managers que querem adotar IA mas nao sabem por onde comecar
  - ~400 milhoes de SMEs globalmente (OECD)
  - Fonte: OECD SME Report 2025, McKinsey State of AI
- **DOR QUANTIFICADA:**
  - 74% das empresas lutam para escalar IA (McKinsey)
  - 65% usam IA mas sem direcao clara
  - Tempo gasto: 2-5 horas pesquisando ferramentas antes de decidir
  - Directories tem 500+ ferramentas, overload de opcoes
- **MARKET SIZE:**
  - TAM: AI tools market ~$11.78B (2026)
  - SAM: SMB AI tools ~$3-4B
  - Crescimento: 46.7%/ano (AI agents market)
  - Fonte: Statista, nucamp.co
- **CONCORRENTES:**
  | Ferramenta | Preco | Limitacao |
  |------------|-------|-----------|
  | Futurepedia | Free | Directory passivo, 4000+ tools = overload |
  | FindMyAITool | Free | Sem personalizacao, busca generica |
  | Insidr.ai | Free | Lista estatica, sem recomendacao |
- **NOSSA VANTAGEM:** Quiz ATIVO que entende o problema -> recomendacao personalizada (nao lista generica)
- **SCORE:** 43/50
  - Pain: 9/10 (x3) = 27
  - Pay: 6/10 (x3) = 18 (freemium model)
  - Build: 8/10 (x2) = 16
  - Diff: 7/10 (x1) = 7
  - Market: 9/10 (x1) = 9
  - Base: 38 + Mercado: +0 + Validacao: +5
- **Verdict:** BUILD
- **CROSS-REFERENCE:**
  - OECD: SME AI adoption baixa apesar de beneficios comprovados
  - McKinsey: 74% struggle to scale = precisam de direcao
  - Forbes: "2026 marks end of simple chatbot"

### 2. Vibe Prompt Hub

- **Source:** Gartner, Wikipedia (Vibe Coding), DEV.to, Indie Hackers
- **O QUE FAZ:** Biblioteca curada de prompts especificos para vibe coding com ferramentas como Cursor, Windsurf, Bolt, Lovable
- **QUEM PRECISA:**
  - Desenvolvedores indie, non-technical founders, vibe coders
  - Milhoes de devs usando AI-assisted coding
  - Fonte: Gartner, Wikipedia (Vibe Coding)
- **DOR QUANTIFICADA:**
  - 80% of enterprise apps by non-devs by 2026 (Gartner)
  - Prompts genericos = resultados ruins
  - Tempo: 15-30 min testando prompts ate achar um bom
  - "Vibe coding hangover" = devs frustrados com AI-generated code ruim
- **MARKET SIZE:**
  - TAM: Developer tools market ~$15B
  - SAM: AI coding tools ~$2-3B
  - Crescimento: Vibe coding e tendencia emergente (2025-2026)
  - Fonte: Wikipedia, Indie Hackers, Fast Company
- **CONCORRENTES:**
  | Ferramenta | Preco | Limitacao |
  |------------|-------|-----------|
  | VibeCodex | Free | Generico, sem filtros por ferramenta |
  | ChatGPT prompts | Free | Nao especifico para coding tools |
  | GitHub Gists | Free | Desorganizado, sem curadoria |
- **NOSSA VANTAGEM:** Filtros por ferramenta (Cursor vs Windsurf vs Bolt) + categorias + upvotes da comunidade
- **SCORE:** 38/50
  - Pain: 7/10 (x3) = 21
  - Pay: 6/10 (x3) = 18
  - Build: 9/10 (x2) = 18
  - Diff: 6/10 (x1) = 6
  - Market: 7/10 (x1) = 7
  - Base: 35 + Mercado: +0 + Validacao: +3
- **Verdict:** MAYBE
- **CROSS-REFERENCE:**
  - Gartner: 80% apps by non-devs = demanda por bons prompts
  - Wikipedia: Vibe coding e termo oficial agora
