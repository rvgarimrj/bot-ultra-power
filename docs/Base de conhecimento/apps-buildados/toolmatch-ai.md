## Descricao
Quiz interativo que entende o problema/necessidade de negocio do usuario e recomenda ferramentas de IA especificas. Nao e um directory passivo - e uma experiencia personalizada com match score.

## Mercado
- **Track:** Global
- **Score:** 43/50
- **TAM:** 74% das empresas lutam para escalar IA (McKinsey). Directories com 4000+ ferramentas = overload
- **Publico:** SMB owners e solopreneurs querendo adotar IA, managers sem tempo para pesquisar, non-technical founders

## Stack
- Next.js 14 (App Router)
- Tailwind CSS + shadcn/ui
- Neon PostgreSQL + Prisma (tools database)
- Railway (deploy planejado)

## Diferenciais
- Quiz ATIVO (nao lista passiva como directories)
- Recomendacao personalizada baseada no problema real
- Curadoria de qualidade (30-50 ferramentas top vs 4000+ em directories)
- 5 perguntas simples -> 3 recomendacoes com match score
- Database inicial com 30 ferramentas curadas por categoria
