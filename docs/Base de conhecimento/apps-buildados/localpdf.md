## Descricao
Toolkit de PDF 100% client-side usando WebAssembly. Merge, split, compress e rotate de PDFs sem upload para servidores. Privacidade total - arquivos nunca saem do dispositivo.

## Mercado
- **Track:** Global
- **Score:** 43/50
- **TAM:** Mercado global de ferramentas PDF (bilhoes de PDFs processados/ano)
- **Publico:** Usuarios preocupados com privacidade, trabalhadores remotos com contratos, estudantes, qualquer pessoa que nao confia em ferramentas online de PDF

## Stack
- Next.js 15 (App Router)
- React + Tailwind CSS
- pdf-lib (MIT license, 100% client-side)
- Sem backend/database
- Railway (deploy planejado)

## Diferenciais
- 100% processamento no browser (zero uploads)
- Privacidade total - arquivos nunca saem do dispositivo
- Sem instalacao necessaria (vs desktop apps)
- Gratuito (vs Adobe)
- Mais facil que PDFtk
- Funciona com internet lenta (sem upload)
