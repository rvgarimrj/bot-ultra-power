## 1. Metadata Patterns

## 2. Hreflang Configuration

### Basic Multilingual Setup

**Supported Locales:** pt-BR, en-US, es (can expand to fr, zh-CN)

**X-Default:** en-US (primary market)

### Common Mistakes to Avoid

- [ ] Missing x-default tag
- [ ] Self-referencing hreflang missing
- [ ] Using relative URLs instead of absolute URLs
- [ ] Forgetting reciprocal links between pages
- [ ] Canonical conflicts between domains

### Hreflang Code Pattern

```typescript
// In generateMetadata function
alternates: {
  canonical: `${baseUrl}/${locale}${path}`,
  languages: {
    'pt-BR': `${baseUrl}/pt-BR${path}`,
    'en-US': `${baseUrl}/en-US${path}`,
    'es': `${baseUrl}/es${path}`,
    'x-default': `${baseUrl}/en-US${path}`
  }
}
```

## 3. Sitemap Implementation

### Multilingual Sitemap Pattern

Generate URLs for all locales with proper alternates. Each URL should have language alternates.

**Priority:** Same across all locales
**ChangeFreq:** Same across all locales

### Sitemap Code Pattern

```typescript
// sitemap.ts
const locales = ['pt-BR', 'en-US', 'es'];

return routes.flatMap(route =>
  locales.map(locale => ({
    url: `${baseUrl}/${locale}${route}`,
    alternates: {
      languages: Object.fromEntries(
        locales.map(l => [l, `${baseUrl}/${l}${route}`])
      )
    }
  }))
);
```

## 5. Locale-Specific Guidelines

### pt-BR (Brazilian Portuguese)

#### Search Engine
- Primary: Google.com.br
- Mobile-first market (80%+ mobile access)

#### URL Structure
- Use hyphens
- Remove accents: `precos` NOT `precos`
- Use Portuguese words for slugs

**Examples:**
- `/pt-BR/como-criar-roteiros-youtube`
- `/pt-BR/precos`
- `/pt-BR/funcionalidades`
- `/pt-BR/sobre-nos`

#### E-commerce Keywords
- Pix
- Boleto
- Parcelado / em ate 12x sem juros
- Frete gratis
- Entrega rapida

#### Cultural Notes
- Brazilians value installment payments (parcelamento)
- Multiple payment options are expected
- Many users have limited data plans - optimize page size

### es (Spanish)

#### Search Engine
- Google regional variants
- Mobile-first in LATAM (70%+ mobile)

#### URL Structure
- Use hyphens
- Remove accents: `como` NOT `como`
- Use Spanish words for slugs

**Examples:**
- `/es/como-crear-guiones-youtube`
- `/es/precios`
- `/es/caracteristicas`

#### E-commerce Keywords
- Pago contra entrega
- Tarjeta
- Transferencia
- Envio gratis

#### Currency Adaptation
- EUR for Spain
- MXN for Mexico
- ARS for Argentina
- Payment methods vary significantly by country

#### Cultural Notes
- Optimize for slow connections in LATAM
- Compress images aggressively
- Limited data plans are common

### en-US (American English)

#### Search Engine
- Primary: Google.com
- Mix of desktop and mobile

#### URL Structure
- Use hyphens
- Standard English slugs

**Examples:**

## 7. Template Code

### generateMetadata Template

```typescript
// app/[locale]/layout.tsx or page.tsx
import { getTranslations } from 'next-intl/server';

const baseUrl = process.env.NEXT_PUBLIC_BASE_URL || 'https://yourdomain.com';
const locales = ['pt-BR', 'en-US', 'es'] as const;

export async function generateMetadata({
  params
}: {
  params: Promise<{ locale: string }>
}) {
  const { locale } = await params;
  const t = await getTranslations({ locale, namespace: 'Meta' });

  const path = ''; // Adjust per page

  return {
    title: t('title'),
    description: t('description'),

    // Canonical and Hreflang
    alternates: {
      canonical: `${baseUrl}/${locale}${path}`,
      languages: {
        'pt-BR': `${baseUrl}/pt-BR${path}`,
        'en-US': `${baseUrl}/en-US${path}`,
        'es': `${baseUrl}/es${path}`,
        'x-default': `${baseUrl}/en-US${path}`,
      },
    },

    // OpenGraph
    openGraph: {
      title: t('title'),
      description: t('description'),
      url: `${baseUrl}/${locale}${path}`,
      siteName: 'YourBrand',
      locale: locale.replace('-', '_'),
      type: 'website',
      images: [
        {
          url: `${baseUrl}/og-image-${locale}.png`,
          width: 1200,
          height: 630,
          alt: t('title'),
        },
      ],
    },

    // Twitter
    twitter: {
      card: 'summary_large_image',
      title: t('title'),
      description: t('description'),
      images: [`${baseUrl}/og-image-${locale}.png`],
    },
  };
}
```

### sitemap.ts Template

```typescript
// app/sitemap.ts
import { MetadataRoute } from 'next';

const baseUrl = process.env.NEXT_PUBLIC_BASE_URL || 'https://yourdomain.com';
const locales = ['pt-BR', 'en-US', 'es'] as const;

// Define your routes
const routes = [
  '',           // homepage
  '/pricing',
  '/features',
  '/about',
  '/contact',
  '/blog',
];

export default function sitemap(): MetadataRoute.Sitemap {
  const now = new Date().toISOString();

  return routes.flatMap(route =>
    locales.map(locale => ({
      url: `${baseUrl}/${locale}${route}`,
      lastModified: now,
      changeFrequency: route === '' ? 'weekly' : 'monthly' as const,
      priority: route === '' ? 1.0 : 0.8,
      alternates: {
        languages: Object.fromEntries([
          ...locales.map(l => [l, `${baseUrl}/${l}${route}`]),
          ['x-default', `${baseUrl}/en-US${route}`],
        ]),
      },
    }))
  );
}
```

### robots.ts Template

```typescript
// app/robots.ts
import { MetadataRoute } from 'next';

const baseUrl = process.env.NEXT_PUBLIC_BASE_URL || 'https://yourdomain.com';

export default function robots(): MetadataRoute.Robots {
  return {
    rules: [
      {
        userAgent: '*',
        allow: '/',
        disallow: [
          '/api/',
          '/admin/',
          '/_next/',
          '/private/',
        ],
      },
    ],
    sitemap: `${baseUrl}/sitemap.xml`,
  };
}
```

### JSON-LD Component Template

```typescript
// components/seo/json-ld.tsx
import Script from 'next/script';

interface OrganizationSchemaProps {
  name: string;
  url: string;
  logo: string;
  description: string;
  sameAs?: string[];
}

export function OrganizationSchema({
  name,
  url,
  logo,
  description,
  sameAs = []
}: OrganizationSchemaProps) {
  const schema = {
    '@context': 'https://schema.org',
    '@type': 'Organization',
    name,
    url,
    logo,
    description,
    sameAs,
  };

  return (
    <Script
      id="organization-schema"
      type="application/ld+json"
      dangerouslySetInnerHTML={{ __html: JSON.stringify(schema) }}
    />
  );
}

interface SoftwareAppSchemaProps {
  name: string;
  description: string;
  applicationCategory: string;
  price: string;
  priceCurrency: string;
}

export function SoftwareAppSchema({
  name,
  description,
  applicationCategory,
  price,
  priceCurrency,
}: SoftwareAppSchemaProps) {
  const schema = {
    '@context': 'https://schema.org',
    '@type': 'SoftwareApplication',
    name,
    description,
    applicationCategory,
    operatingSystem: 'Web',
    offers: {
      '@type': 'Offer',
      price,
      priceCurrency,
    },
  };

  return (
    <Script
      id="software-app-schema"
      type="application/ld+json"
      dangerouslySetInnerHTML={{ __html: JSON.stringify(schema) }}
    />
  );
}
```
